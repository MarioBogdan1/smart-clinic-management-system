# User Stories

## Doctor
As a doctor, I want to view my appointments so that I can manage my schedule.

As a doctor, I want to create prescriptions so that I can record patient treatments.

## Patient
As a patient, I want to search for doctors so that I can find the right specialist.

As a patient, I want to book appointments so that I can receive medical care.

## Admin
As an admin, I want to manage doctors so that I can maintain the system.

As an admin, I want to add doctors so that patients can book appointments.
